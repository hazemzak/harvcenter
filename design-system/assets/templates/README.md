## What these assets show about the current brand

- **Logo:** Red stylized H with graduation cap and stars
- **Colors:** Navy blue + red + gold
- **Typography:** Mixed Arabic display fonts
- **Layout:** Dense, decorated, inconsistent between posts
- **Photography:** Real teacher photos (good!) with backgrounds removed

## What the new system changes

- Logo: same red H shape, but flat — no cap, no stars, no 3D
- Colors: navy + red + white — no gold
- Typography: Cairo (one font) at consistent sizes
- Layout: clean, consistent templates with breathing room
- Photography: keep the real teacher photos — they're the strongest asset
