# Infographics — infographics.md

Used for: exam tips, subject comparisons, study method breakdowns, results summaries.

---

## Canvas sizes

| Type | Size | Use |
|------|------|-----|
| Instagram carousel | 1080 × 1350 (4:5) | Multi-slide tips, breakdowns |
| Single post | 1080 × 1080 (1:1) | Comparison charts, simple stats |
| Story / wide | 1080 × 1920 (9:16) | Vertical process flows |
| Print / poster | A4 or A3 | Printed study guides for the center |

## Four infographic types

### 1. Comparison
Side-by-side comparing two approaches, subjects, or study methods.

- Split layout: left vs. right
- Red header bar for option A, navy for option B
- 3–5 comparison points, each a single line
- Example: "ثانوية عامة vs. بكالوريا — إيه الفرق؟"

### 2. Process flow
Step-by-step guide (exam prep timeline, enrollment steps, study plan).

- Vertical flow, numbered steps
- Each step: number (Cairo Bold, red, 32px) + title + 1-line description
- Connected by a thin red vertical line
- Maximum 5 steps

### 3. Big number
A single stat made dramatic.

- One number: Cairo Bold, 96px, red, centered
- Context line below: Cairo Regular, 18px, navy
- Example: "92%" + "نسبة نجاح طلاب Harv في ثانوية عامة 2026"

### 4. Tips / checklist
Study tips, exam-day advice, subject-specific tricks.

- Carousel format (one tip per slide)
- Slide 1: hook ("5 حاجات تعملهم ليلة الامتحان")
- Slides 2–6: one tip per slide, numbered
- Final slide: CTA or summary

## Layout rules

- Off-white background for information-heavy infographics
- Navy background for single-stat or dramatic posts
- Maximum 3 text sizes per infographic
- Cairo only — no decorative fonts
- Icons: Lucide, 1.5 stroke, ink-600 or red
- No decorative borders, no gold accents, no 3D elements

## Anti-patterns

- Pie charts with more than 4 slices
- Infographics with more than 50 words per slide
- Generic study advice that isn't tied to a specific subject or exam
- Stock illustrations of books, pencils, or graduation caps
- Watermark-heavy designs that obscure content

## Generation workflow (for Claude/AI tools)

1. Define the single message of the infographic
2. Pick the type (comparison / process / big number / tips)
3. Write the content first — plain text, no design
4. Apply the layout from this guide
5. Check: could a student understand this in 5 seconds? If not, simplify.
