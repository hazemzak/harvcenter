# Instagram — social-instagram.md

Instagram is Harv's primary channel. Most students find us here. Every post must look like it belongs to the same family.

---

## Aspect ratios

| Format | Ratio | Pixels | When to use |
|--------|-------|--------|------------|
| Feed post (default) | 4:5 | 1080 × 1350 | Teacher cards, announcements, results |
| Feed post (square) | 1:1 | 1080 × 1080 | Schedule tables, QR codes, simple graphics |
| Story / Reel cover | 9:16 | 1080 × 1920 | Stories, reel covers, countdowns |
| Carousel slide | 4:5 | 1080 × 1350 | Multi-slide teaching content, lineup |

Default to **4:5** unless there's a reason not to.

---

## Design rules — every post

### Color
- Background: navy (#1A2744) or warm off-white (#FAFAF8) — pick one per post
- Primary accent: red (#D42027) — used for the H mark, key info highlights, dividers
- Text: white on navy; navy on off-white
- Never gold. Never gradients. Never multi-color backgrounds.

### Typography
- **Cairo Bold** for headlines (24–32px equivalent)
- **Cairo SemiBold** for subheads and teacher names (18–20px)
- **Cairo Regular** for body text, dates, details (14–16px)
- All text sentence case — no Title Case, no ALL CAPS except overline kickers ≤3 words
- RTL alignment for Arabic content

### Logo placement
- H mark: top-left corner or bottom-center
- Size: 40–60px height
- Always flat, always red on navy or red on white
- Never add effects, glow, or borders to the mark

### Layout
- 60px minimum padding from edges
- Maximum 3 text sizes per post
- Teacher photo: right-aligned or centered, background-removed, waist-up crop
- Clear visual hierarchy: what → who → when → where

### Elements to never use
- Gold ornamental borders or frames
- 3D renders of the logo
- Harvard University campus imagery
- Graduation cap on the H
- Stars around the logo
- More than 3 emoji per caption
- Decorative clock/calendar icons (use text for time/date)
- AI-generated backgrounds or video

## Post templates

### 1. Teacher announcement card (most common)

The bread and butter. Used for: announcing a class, a review session, or a new teacher.

**Layout (4:5, navy background):**
```
┌─────────────────────────────────┐
│  [H mark]              [Flag]  │  ← Flag only for language subjects
│                                 │
│   Subject name                  │  ← Cairo Bold, white, 28px
│   Grade level                   │  ← Cairo Regular, red, 16px
│                                 │
│          [Teacher photo]        │  ← Background-removed, centered
│                                 │
│   Teacher name                  │  ← Cairo SemiBold, white, 20px
│                                 │
│  ┌─────────────────────────┐   │
│  │ Day — Date — Time       │   │  ← Red background bar, white text
│  └─────────────────────────┘   │
│                                 │
│  Brief description (1 line)     │  ← Cairo Regular, white/70%, 14px
│                                 │
└─────────────────────────────────┘
```

**Rules:**
- One teacher per card
- Subject and grade first, teacher name second — the student scans for their subject, not the teacher's name
- Date format: يوم الخميس 11/6 — الساعة 6:30 م
- Maximum one red accent bar for timing
- Flag motif (small, corner) only for language subjects — French, Italian, German, English
- No decorative backgrounds behind the teacher photo — just the flat navy

### 2. Exam review session card

Same as teacher card but with added info about what's covered.

**Additional elements:**
- "مراجعة ليلة الامتحان" as the overline (Cairo SemiBold, red, 14px)
- A 1-line content description: "أهم 20 سؤال متوقع" or "المنهج كامل في حصة"
- No emoji in the card itself — save them for the caption

### 6. World Cup-style teacher lineup

A creative concept for showcasing the full teaching staff as a football team.

**Layout (4:5, navy background):**
```
┌─────────────────────────────────┐
│  [H mark]       STARTING XI     │  ← Overline in English works here
│                                 │
│   تشكيلة Harv ٢٠٢٧             │  ← Cairo Bold, white, 28px
│                                 │
│        [Teacher]                │  ← Formation layout:
│       [T]  [T]  [T]            │     3 rows mimicking
│      [T]  [T]  [T]  [T]       │     a football formation
│        [T]  [T]  [T]           │
│                                 │
│   Each teacher: small circular  │
│   photo + name + subject below  │
│                                 │
│   معاك للنهاية                  │
│                                 │
└─────────────────────────────────┘
```

**Rules:**
- Teacher photos: circular crop, 80px diameter, white 2px border
- Name below each photo in Cairo SemiBold, white, 12px
- Subject below name in Cairo Regular, red, 10px
- Formation layout should loosely mirror a real football formation
- Can be used as a single post or as a carousel (slide 1 = formation, slides 2+ = individual teacher stats cards)
- Works as a seasonal concept — can be adapted for any sports event

**Carousel version (World Cup teacher lineup):**
- Slide 1: The full formation (as above)
- Slide 2–N: Individual "player card" per teacher — portrait photo, name, subject, years teaching, fun stat ("أكتر مدرس طلابه بيجيبوا A")
- Final slide: CTA — "عايز تنضم للتيم؟ الحجز مفتوح."

## Carousel structure

For teaching content, tips, or multi-part announcements:

| Slide | Purpose | Background |
|-------|---------|-----------|
| 1 — Hook | Bold statement or question | Navy |
| 2–N — Content | One idea per slide, scannable | Off-white |
| Final — CTA | Call to action + contact | Navy |

**Rules:**
- Maximum 7 slides (attention drops after that)
- One idea per slide, large type
- Consistent placement of slide number (top-right, Cairo Regular, 12px)
- Progress dots or "2/6" indicator on each slide
- Final slide always has WhatsApp number or action

## Stories and Reels

### Stories
- 9:16, navy or off-white background
- Use for ephemeral content: "المراجعة بدأت," classroom photos, quick polls
- Always add a location sticker (مدينة نصر)
- Use the question/poll stickers — engagement, not broadcasting

### Reel covers
- 9:16, navy background
- Title in Cairo Bold, centered
- H mark bottom-center
- No cluttered text — 3–5 words maximum
- If it's a filmed reel: use a still frame from the actual footage, not a designed cover

### What we never post
- AI-generated videos (Higgsfield or similar)
- Stock footage of "students studying"
- Reels with trending audio that doesn't relate to education
- Memes that don't land with the 14–18 audience
- Posts in English-only (always Arabic-first or bilingual)

## Caption template

```
[One-line description of what this is]

[Key details: subject, teacher, date, time]

[1 specific claim or number]

للحجز أو الاستفسار:
📲 01064949395
📍 مدينة نصر — المنطقة التاسعة

#Harv #هارف #ثانوية_عامة #[subject_hashtag]
```

**Hashtag rules:**
- Maximum 8 hashtags
- Always include: #Harv #هارف
- Add subject-specific: #فيزياء #كيمياء #فرنساوي etc.
- Add exam-specific during season: #ثانوية_عامة #بكالوريا
- Never use: #تفوق #نجاح #مستقبل (generic, no reach)
